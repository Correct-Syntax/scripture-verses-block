Very simple plugin to add Scripture verses to your WordPress posts and pages.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Create a new post ( or page! ).
1. Add Scripture verses to your post by clicking the '+' icon and selecting "Scripture Verses".
1. Each block can have multiple sections.
1. Press publish!
